# The Legend of Zelda - The ShadowGazer 3DS

This game includes:

	- PP2D Library.
	- Resources (images and fonts) loaded from romfs
	- Simple save for game data
	- Simple code that can be used as reference
	- SplashScreen 
	- Sounds
	- .3dsx, .elf, .cci, .3ds, .cia compiling
